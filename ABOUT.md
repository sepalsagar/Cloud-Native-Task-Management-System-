# About

Cloud-Native Task Management System is a production-ready MERN application for collaborative task planning and execution.

## Short Description

Scalable full-stack task manager with JWT authentication, role-based access control, clean REST APIs, and Docker-based cloud-native deployment.

## Core Highlights

- React frontend with board-based task workflows.
- Node.js/Express backend with MongoDB persistence.
- JWT auth + RBAC (`admin`, `member`) and task-level access policy.
- Dockerized services with `docker-compose` orchestration.
